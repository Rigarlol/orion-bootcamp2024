FUNÇÕES FUNCIONAIS

Retornar a bio a partir de um ID

                            
function returnBioPerson(id: number): string {
    let BioPerson = list.find(item => item.id === id);
    
    return BioPerson ? BioPerson.bio : 'Bio não econtrada para o id indicado';
}

FUNÇÕES IMPERATIVAS

Retornar a bio a partir de um ID

                            
function returnBioPersonImperative(id: number): string {
    for (let i = 0; i < list.length; i++) {
        if (list[i].id === id) {
        return list[i].bio;
        }
    }
    return 'Bio não econtrada para o id indicado';
}

console.log('Bio:', returnBioPersonImperative(3))

//resposta no console:

Bio: Nikola Tesla foi um inventor, engenheiro eletrotécnico e engenheiro mecânico sérvio, 
mais conhecido por suas contribuições ao projeto do moderno sistema de fornecimento de eletricidade em corrente alternada.
